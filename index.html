About

THEFAXMINATOR

TheFaxminator is an alias created so I could play COD. The alias have now expanded to become a website. The main purpose of the website is to try to make available different types of tools that people/companies can utilise if they find them useful. There is no separate server and the tools will not collect any data.

Information

What's on the Website

The website currently offers a tin validator and information about the supported TIN's, with more planned as it's under development:

Global TIN Validator TIN's by Country

The Global TIN Validator lets you validate any tax identification number against regular expression rules. Though a number of countries do have check digit algorithms as part of the TIN structure, this is not included here. The formats supported by THEFAXMINATOR is shown in the TIN information cards. The validator will instantly flag valid or invalid TIN's.

TIN's by Country is a growing reference library where each supported country has a dedicated profile covering its TIN types, formats, issuing authority, and some other facts.

Data & Architecture

Single Source of Truth

All TIN data — descriptions, format rules, validation logic, and country metadata — lives in dedicated per-country modules. Both the validator and the country reference pages read from the same source, so information is always consistent across the platform. New countries are added by creating a single file.
